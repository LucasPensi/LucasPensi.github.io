O que é programação?

Programação é o processo de criação de um conjunto de instruções que dizem ao computador como realizar uma tarefa. A programação ajuda no desenvolvimento da capacidade de comunicação e interação entre as pessoas. Quando aprendemos programação, podemos fazer uso de novas tecnologias para criar algo novo e nos expressar plenamente por meio dessa criação.
Hoje em dia ela está em todos os lugares, sites, redes sociais, aplicativos, sistemas de empresas, entre outros e é tão importante quanto ler e escrever. Apesar disso, a linguagem de programação não surgiu do nada, teve uma história por trás disso e muitas pessoas que influenciaram nela.

A Evolução Da Programação

A primeira linguagem de programação foi criada em 1843, por uma mulher chamada Ada Lovelace, ela foi a primeira a desenvolver um algoritmo para implantação em um computador, a máquina de análise de Charles Babbage . No algoritmo feito por Ada, a máquina seria capaz de calcular sequências de Bernoulli, e este é considerado por muitos historiadores e programadores como o primeiro programa criado na história. Curiosamente a máquina analítica de Babbage é considerada também como o ponto de partida para os computadores que conhecemos atualmente.

Durante a Segunda Guerra Mundial, foi criada, por Konrad Zuse, a primeira e mais próxima ao que temos atualmente. Chamava-se Plankalkül e foi desenvolvida para que o exército nazista enviasse mensagens codificadas a suas tropas.

Décadas de 60 á 80

A partir da década de 60 e 70 surgiram algumas linguagens, tipo a C, uma das primeiras linguagens de sistema, também surgiram pascal, simula e etc.
A internet faz parte desses tempos, foi criada em 1969, tinha como função interligar laboratórios de pesquisa.
Os anos 80 foram marcante, pois as empresas buscaram investir nas linguagens criadas das décadas de 60 e 70. Um exemplo disso é a C++, uma evolução da linguagem C, com nível intermediário, orientada a objetos. Ela foi criada por Bjarne Stroustrup em 1983. Seus usos foram para o desenvolvimento de aplicações comerciais, produção de jogos, etc. Adobe, Microsoft (Internet Explorer), Google (Google Chrome), Mozilla Firefox são exemplos em que ela foi utilizada.

Linguagens De Programação

Java- Nos anos 90 foram criadas a Java e Javascript, a Java é definida como linguagem orientada a objetos, ela faz com que os desenvolvedores não precisem escrever o programa mais de uma vez e que o façam em qualquer dispositivo.


HTML- Também nos anos 90, o HTML foi criado, utilizado na construção apenas de páginas na Web.


CSS- No final da década de 90, CSS foi criado, é um mecanismo para adicionar estilo (cores, fontes, espaçamento, etc.) a um documento web.
